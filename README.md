# rhine: Turn-based War Strategy

The game will have you play full military tactics and planning.

Features:
* Millitary Organizations
* Realistic simulation such as supply lines, terrains, etc.
* mulitplayer hotseat
* single player against AI
* **NO INSTALL REQUIRED**
